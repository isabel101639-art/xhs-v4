# xhs_v4 账号同步爬虫服务接入说明

- 版本：`v1`
- 日期：`2026-04-11`
- 适用范围：`xhs_v4/crawler_service`

## 1. 目标

给 `xhs-v4` 提供一个独立的账号同步服务，用于接收报名人账号主页列表，返回：

1. `accounts`
2. `posts`
3. `snapshots`

主站 Worker 收到这些数据后，会直接导入账号看板，并自动回流到“报名人持续跟踪”统计。

## 3. 当前支持的 Provider

### `mock`

默认模式，方便联调。

特点：

1. 不依赖真实小红书页面
2. 可直接返回模拟账号、笔记、快照
3. 适合先打通 `xhs-v4` 主站自动化链路

### `playwright_xhs`

预留真实抓取位。

特点：

1. 支持用 `Playwright` 打开账号主页
2. 内置一组默认候选选择器和链接型兜底逻辑
3. 可通过环境变量覆盖页面选择器
4. 配套提供登录态保存脚本和页面调试脚本

## 5. 本地启动

### 方式 A：先用 mock 联调

1. 执行 `crawler_service/scripts/bootstrap_venv.sh`
2. 执行 `scripts/run_crawler_dev.sh`
3. 设置主站自动化中心：
   - `账号同步模式 = remote`
   - `账号同步 API URL = http://127.0.0.1:8081/xhs/account_posts`

### 方式 B：切到 Playwright

1. 设置 `CRAWLER_PROVIDER=playwright_xhs`
2. 先准备登录态：
   - 执行 `python crawler_service/scripts/save_xhs_storage_state.py`
   - 完成登录后按回车
   - 把生成的 state 路径填到 `PLAYWRIGHT_STORAGE_STATE_PATH`
3. 如果抓取结果不理想，先调试页面：
   - 设置 `XHS_DEBUG_PROFILE_URL`
   - 执行 `python crawler_service/scripts/debug_xhs_profile.py`
   - 查看 `XHS_DEBUG_OUTPUT_DIR` 下的截图、HTML 和调试 JSON
4. 按真实页面逐步配置：
   - `XHS_POST_CARD_SELECTOR`
   - `XHS_POST_LINK_SELECTOR`
   - `XHS_POST_TITLE_SELECTOR`
   - `XHS_PROFILE_NAME_SELECTOR`
   - `XHS_FOLLOWER_COUNT_SELECTOR`
5. 再启动 crawler 服务

## 6. 脚本

### 保存登录态

```bash
cd root/xhs_v4
crawler_service/scripts/bootstrap_venv.sh
PLAYWRIGHT_STORAGE_STATE_OUTPUT=./crawler_service/.state/xhs_storage_state.json \
./crawler_service/.venv/bin/python crawler_service/scripts/save_xhs_storage_state.py
```

### 调试账号主页

```bash
cd root/xhs_v4
XHS_DEBUG_PROFILE_URL="https://www.xiaohongshu.com/user/profile/xxx" \
PLAYWRIGHT_STORAGE_STATE_PATH=./crawler_service/.state/xhs_storage_state.json \
./crawler_service/.venv/bin/python crawler_service/scripts/debug_xhs_profile.py
```

## 7. Docker Compose

仓库根目录 `docker-compose.yml` 已增加 `crawler` 服务。

如果使用容器内网地址，主站建议配置：

- `CREATOR_SYNC_API_URL=http://crawler:8081/xhs/account_posts`

## 8. 建议推进顺序

1. 先用 `mock` 跑通主站账号同步任务
2. 再切 `playwright_xhs`
3. 先生成登录态 `storage state`
4. 先跑调试脚本，看截图和 HTML
5. 先只抓标题、链接、点赞/评论这些基础字段
6. 页面选择器稳定后，再补粉丝、曝光、更多统计

## 9. 备注

当前 `playwright_xhs` 已经有：

1. 默认候选选择器
2. 链接型兜底提取
3. 登录态保存脚本
4. 页面调试脚本

但真实选择器、登录态和页面结构适配仍需要根据你实际账号页面校准。
