## 3. 已准备好的联调脚本

### 创建并安装虚拟环境

```bash
./scripts/bootstrap_venv.sh
```

### 启动本地开发服务

```bash
./scripts/run_local_dev.sh
```

### 启动本地 crawler 服务

```bash
./crawler_service/scripts/bootstrap_venv.sh
./scripts/run_crawler_dev.sh
```

### 运行 crawler 热点接口冒烟

```bash
./crawler_service/.venv/bin/python crawler_service/scripts/smoke_trends.py
```

### 探测真实小红书热点 / 爆款内容

```bash
PLAYWRIGHT_STORAGE_STATE_PATH=./crawler_service/.state/xhs_storage_state.json \
CRAWLER_PROVIDER=playwright_xhs \
XHS_PROBE_KEYWORDS="脂肪肝,肝纤维化" \
XHS_PROBE_TREND_TYPE=note_search \
./crawler_service/.venv/bin/python crawler_service/scripts/probe_xhs_trends.py
```

### 先验证登录态是否可用

```bash
PLAYWRIGHT_STORAGE_STATE_PATH=./crawler_service/.state/xhs_storage_state.json \
CRAWLER_PROVIDER=playwright_xhs \
XHS_VERIFY_KEYWORD="脂肪肝" \
./crawler_service/.venv/bin/python crawler_service/scripts/verify_xhs_login_state.py
```

### 一次跑完整 crawler 探测

## 4. 推荐联调顺序

在一台能联网的机器上，建议按下面顺序执行：

1. 复制 `.env.example` 为 `.env`
2. 填写 `SECRET_KEY`、`DEEPSEEK_API_KEY`
3. 执行 `./scripts/bootstrap_venv.sh`
4. 执行 `./crawler_service/scripts/bootstrap_venv.sh`
5. 先启动 `./scripts/run_crawler_dev.sh`
6. 再启动 `./scripts/run_local_dev.sh`
7. 本地联调建议设置：
   - `INLINE_AUTOMATION_JOBS=true`
   - `CREATOR_SYNC_FETCH_MODE=remote`
   - `CREATOR_SYNC_API_URL=http://127.0.0.1:8081/xhs/account_posts`
   - `HOTWORD_FETCH_MODE=remote`
   - `HOTWORD_API_URL=http://127.0.0.1:8081/xhs/trends`
   - `HOTWORD_TREND_TYPE=note_search`
   - `HOTWORD_PAGE_SIZE=20`
   - `HOTWORD_MAX_RELATED_QUERIES=20`
8. 访问：
   - `http://127.0.0.1:5000/healthz`
   - `http://127.0.0.1:5000/admin`
   - `http://127.0.0.1:5000/automation_center`
