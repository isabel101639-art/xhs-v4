
### 启动本地开发服务

```bash
./scripts/run_local_dev.sh
```

### 运行最小冒烟检查

```bash
./.venv/bin/python scripts/smoke_check.py
```

## 4. 推荐联调顺序

在一台能联网的机器上，建议按下面顺序执行：

1. 复制 `.env.example` 为 `.env`
2. 填写 `SECRET_KEY`、`DEEPSEEK_API_KEY`
3. 执行 `./scripts/bootstrap_venv.sh`
4. 执行 `./scripts/run_local_dev.sh`
5. 访问：
   - `http://127.0.0.1:5000/healthz`
   - `http://127.0.0.1:5000/admin`
   - `http://127.0.0.1:5000/automation_center`
