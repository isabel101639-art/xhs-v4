## 1. 当前收口范围

本轮只收口 4 条主线，不再扩新模块：

1. 文案生成质量
2. 图片生成质量
3. 数据分析可用性
4. 自动化数据接入准备

## 2. 已完成项

### 2.1 文案质量

1. DeepSeek `deepseek-v4-pro` 思考模式链路已适配
2. 文案生成已支持：
   - 规划 Agent
   - 写作 Agent
   - 去模板味改写
3. 本地 fallback 已增加：
   - 标题池补强
   - 正文过短修复
   - 版本差异化修复
   - 质量分排序

### 2.2 图片质量

1. 图片接口线上联调已成功
2. 图片 Agent 已支持：
   - 路线分析
   - 参考图分析
   - 封面适配度评分
   - 自动 fallback 到更适合封面的风格
3. 预览素材、图文套组、真实图片任务已开始共用一套风格决策

### 2.3 任务与留痕

1. “我的报名”已升级成“我的任务”
2. 任务页已支持：
   - 状态漏斗
   - 优先处理建议
   - 任务筛选
   - 已选策略摘要
3. 提交前策略选择已可创建草稿 `Submission`
4. 生成路线、图片方案、参考图方案都已进入策略留痕

### 2.4 数据分析

1. 数据分析页已支持：
   - 任务漏斗
   - 策略留痕表
   - 策略结论摘要
2. 标题 Skill / 图片 Skill / 组合打法已统一成中文标签
3. 周报 / 月报 / 复盘导出已包含：
   - 任务漏斗
   - 策略结论

### 2.5 回归

1. `scripts/smoke_check.py` 当前已通过
2. 新增覆盖：
   - 策略草稿预保存
   - 任务漏斗
   - 策略结论摘要
   - 图片决策一致性
   - CSV 表格文本热点解析

## 4. 上线前必查项

上线后可以直接跑：

`./.venv/bin/python scripts/post_deploy_check.py --base-url https://你的域名 --username 后台账号 --password 后台密码`

如果本机证书链有问题，可补 `--insecure`；如果只是先验后端模型链路、还没发前端，可以先补 `--skip-ui-checks`。

### 4.1 模型与图片

1. `DeepSeek v4-pro` 在线文案联调通过
2. `volcengine_ark` 图片联调通过
3. 线上自动化中心中：
   - 文案模型预览正常
   - 图片接口预览正常

### 4.2 数据与任务

1. 任务页能正常查看任务状态
2. 报名成功页能：
   - 生成文案
   - 预览图片方案
   - 发起真实图片任务
   - 提交小红书链接
3. 数据分析页能正常出统计
4. `release-manifest` 能正常返回当前版本指纹

### 4.3 自动化

1. 热点导入能吃 JSON
2. 热点导入能吃 CSV / 表格文本
3. 外部 API 接入后能完成：
   - 导入预览
   - 标准化
   - 候选话题生成
   - 热点分流
